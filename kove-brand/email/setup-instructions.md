# Email signature — setup

Two signatures: **Kove full** (new emails) and **Kove short** (replies/forwards).

1. Open Gmail → Settings (gear) → **See all settings** → **Signature**.
2. Click **Create new** → name it `Kove full`.
3. Open `signature-full.html` in a browser, select the signature block, copy it, and paste into the Gmail signature box. *(Or use “Insert HTML” if your client supports it.)*
4. Replace `[phone]` with your real number. Update the `tel:+64` link too.
5. Under **Signature defaults**, set `Kove full` for **new emails**.
6. Repeat steps 2–3 with `signature-short.html`, name it `Kove short`, set it for **replies/forwards**.

**Colours used:** name `#0B0D0E` · Kove link `#F59E0B` · body `#1a1a1a` · muted `#6B7280` / `#9CA3AF`.
Signatures are table-based and inline-styled so they survive Gmail / Outlook / Apple Mail rendering.
